## Легенда оценок
- **L (Вероятность):** 1-очень низкая, 2-низкая, 3-средняя, 4-высокая, 5-очень высокая
- **I (Влияние):** 1-минимальное, 2-низкое, 3-среднее, 4-высокое, 5-критическое
- **Risk = L × I**

## Стратегии обработки рисков
- **Снизить:** Внедрение контролей для уменьшения вероятности/влияния
- **Принять:** Риск принят, мониторинг без активных мер
- **Избежать:** Изменение архитектуры для исключения риска
- **Перенести:** Передача риска третьей стороне (страхование)
